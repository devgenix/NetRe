# NetRe

NetRe is a simple python script that allows you to find networks your pc has connected to and their passwords.

## How to use

### Windows
1. Open your terminal (Command Prompt or PowerShell).
2. Run the script using:
   ```bash
   python NetRe.py
   ```
3. Follow the prompt and click Enter to run.

### macOS
1. Open your Terminal.
2. Run the script using:
   ```bash
   python3 NetRe.py
   ```
3. Follow the prompt and click Enter to run.
**Note:** macOS may prompt you for your Keychain password to allow the script to retrieve the Wi-Fi passwords.
